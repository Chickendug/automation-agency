# ☀️ WAKE UP — START HERE

Everything is built. **30 minutes** and you're dialing.

## Step 1: Start the app (2 min)

```powershell
cd C:\Users\jeden\Projects\automation-agency
.\scripts\setup.ps1
npm run dev
```

Open **http://localhost:3000/start** ← your morning checklist

## Step 2: Fill in YOUR info (3 min)

Go to **Settings** and set:

- Your name
- Your phone (caller ID / callback)
- Your email
- Agency name
- Default niche: `HVAC / plumbing` (or pick yours)
- Default city: `Your City, ST`

## Step 3: Set up payments (15 min, once)

Go to **Payments** → create free **Stripe** account → make Payment Links for setup + monthly → paste links → Save.

Full guide: `launch-kit/PAYMENTS-GUIDE.md`

## Step 4: Open the dial script (1 min)

Go to **Dial Script** — fullscreen teleprompter on second screen.

## Step 5: Start calling (rest of day)

1. **Call Mode** — dial sample leads or find new ones
2. **Dial Script** — read on every call
3. Interested → **Clients** → **Collect $** → copy pay email/SMS with Stripe link
4. **Do NOT build until setup fee is paid** → then Playbooks
