# Day 1 Checklist — Automation Agency

## Before you dial (30 min)

- [ ] Run `.\scripts\setup.ps1` if first time
- [ ] `npm run dev` → open http://localhost:3000/start
- [ ] Settings: name, phone, email, niche, city
- [ ] Sign up: Twilio + Make.com (Toolkit page)
- [ ] Read dial script out loud 3 times: `/dial`

## Morning block (lead gen)

- [ ] Find 50–100 leads (Find Leads) OR use 10 sample leads
- [ ] Export CSV backup (Leads → Export)

## Power dial block (8am–12pm)

- [ ] Open `/dial` on second screen
- [ ] Open `/leads/call` on main screen
- [ ] Target: **80–100 dials**
- [ ] Log every outcome
- [ ] Goal: **3 interested**

## Afternoon

- [ ] Text/call interested leads — book demos
- [ ] Run demo using Scripts → Demo script
- [ ] Send proposal same day

## Do NOT do today

- ❌ Build custom software
- ❌ Learn every Make module
- ❌ Change your niche mid-day
- ❌ Skip logging calls

## End of day

- [ ] Update Clients pipeline
- [ ] Note tomorrow's callbacks
- [ ] Same niche + same city tomorrow
